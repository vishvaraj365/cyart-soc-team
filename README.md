# Week 2 – SOC Alert to Response Workflow

## Overview
This project demonstrates the complete Security Operations Center (SOC) workflow from alert generation to incident response and reporting. It includes both theoretical knowledge and practical implementation of SOC processes.

## Objectives
- Understand alert prioritization
- Classify security incidents using MITRE ATT&CK
- Perform alert triage and investigation
- Document incident response procedures
- Simulate a real-world security incident

## Topics Covered
1. Alert Priority Levels  
2. Incident Classification  
3. Incident Response Lifecycle  
4. Alert Management  
5. Alert Triage  
6. Threat Intelligence  
7. Evidence Preservation  
8. Capstone Project

## Capstone Project Summary
A simulated attack was performed on a vulnerable system. The attack was detected, analyzed, and contained using SOC procedures. The incident was documented with proper response actions and recommendations.

## Conclusion
This project demonstrates the complete SOC workflow and highlights the importance of structured incident response and documentation in cybersecurity operations.
